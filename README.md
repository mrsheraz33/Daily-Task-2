# DOM Practice - Fetch Posts API

## Project Overview
This project is a simple JavaScript DOM practice using Fetch API.

### Task Completed Today
- Fetch data from API
- Display post titles in a list
- Show loading message while fetching data
- Handle network errors

### Features
- Uses Fetch API with async/await
- Displays first 8 posts from API
- Dynamically creates list items in DOM
- Error handling with try/catch
- Loading state for better user experience

### How to Use
1. Open `index.html` in a browser
2. Click the button
3. Post titles will appear in the list
